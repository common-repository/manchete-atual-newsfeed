Newsfeed, Listagem de notícias, personalizavel, disponibilizado com conteúdos do site Manchete Atual.

== Description ==

Agora pode ter as notícias mais recentes e/ou mais importantes no seu site ou blog. Conteúdos disponibilizados pelo site Manchete Atual.
O Newsfeed usa design responsivo para se adaptar ao seu site e pode ser personalizado o número de entradas a mostrar, até um máximo de 10 notícias, bem como o tipo de notícias a listar: Mais Recentes ou Destaques Manchete Atual.

Nota:
Este plugin faz uso do sistema de cache do WordPress (5 min) para evitar pedidos constantes ao servidor do Manchete Atual.

== Installation ==

1. Descarregue o plugin no menu plugins do WordPress ou coloque o conteúdo do arquivo ZIP na pasta `/wp-content/plugins/`.
2. Ative o plugin no menu plugins do WordPress
3. Configure o widget com valores à sua escolha: título, cores, fonte e categoria a mostrar.

== Frequently asked questions ==

= Que notícias posso mostrar em meu site com o plugin Newsfeed? =
Estão disponíveis os seguintes tipos: Notícias Recentes; e Notícias em Destaque

= Posso combinar os dois tipos de notícias? =
Não.
